About BurnNote

BurnNote is a simple and secure note-taking web application designed to make your notes ephemeral and safe.

Create ephemeral notes

Your notes are deleted automatically after being read.

Set expiration dates

Control how long your notes remain accessible.

Built for simplicity

Developed with Flask and SQLite for efficiency and ease of use.

Secure encryption

Uses AES-GCM encryption to protect your notes.

Python and SQLAlchemy

Developed with Python and the SQLAlchemy ORM.

Marshmallow integration

Uses Marshmallow for data validation and serialization.

RESTful API

Provides an API for programmatic access to notes.
